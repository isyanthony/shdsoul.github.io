CSS(二):grid布局

发表于 2022-01-08 分类于 CSS
本文字数： 2.4k 阅读时长 ≈ 2 分钟
CSS中十分常用的响应式布局方案，grid布局，自认为比flex布局更为可控更加舒适.

引言，起初了解到grid布局是在BootStrap中，比较著名的栅格布局，也就是grid布局，十分的好用。通俗来讲就是将网页的长宽分为n个等宽高的区域，我们可以通过css的编写来让一个元素占有一定数量的区域从而达到布局的效果，并且还具有响应式的特点，十分的好用。

属性

使用grid布局

​ grid布局与flex布局的相似之处在于，其主要操作布局container元素容器中的第一级子元素，而不会改变其第二级子元素.

注意:

1. 默认为块级元素，值为inline-grid时为行内块元素.

2. 设为网格布局以后，容器子元素（项目）的float、display: inline-block、display: table-cell、vertical-align和column-*等设置都将失效。
